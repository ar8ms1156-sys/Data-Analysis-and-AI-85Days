* **حجم الطلبات والكميات (Quantity):** لمعرفة الأصناف والمنتجات الأكثر استهلاكاً وحركة.
* **أسعار البيع (Price):** لمراقبة استقرار أسعار الأصناف الفاخرة والأساسية مثل أرز الشعلان ومياه نوفا.
* **إجمالي الفواتير (Total Revenue):** لتقييم العوائد المالية الإجمالية للمشروع والتي بلغت **8,155.00 ريال**.

💡 **ملاحظة المحلل:** تم إنشاء وتصميم الجداول المحورية (Pivot Tables) والرسوم البيانية وتنسيق الألوان بالكامل داخل ملف الإكسل بما يتوافق مع الهوية البصرية لتطبيق "نعناع"، مع ربط المقسمات (Slicers) لتسهيل قراءة الأرقام ودعم اتخاذ القرار بشكل تفاعلي وسريع.
